# Gerador de senhas da Monyk a Bruxa

Gerador de senhas aleatórias desenvolvido em HTML, CSS e JavaScript. Permite personalizar o tamanho (de 1 a 20 caracteres) e os tipos de caracteres utilizados (letras maiúsculas, minúsculas, números e símbolos), além de exibir a força da senha com base na entropia.

## Como usar

Abra o arquivo `index.html` no navegador.

## Arquivos

- `index.html` - Estrutura da página
- `style.css` - Estilos
- `main.js` - Lógica de geração e classificação de senhas
- `cadeado.png` - Imagem do cabeçalho
